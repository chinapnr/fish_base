## fishbase 简介

这是我们在学习 python 过程中积累的一个函数库，主要用于自己项目的学习使用，希望对 python 的初学者等有所帮助。

---

目前主要分为：

* fish_common 基本函数包
* fish_file 文件处理函数包
* fish_csv csv 函数包
* fish_logger 日志记录函数包

详细帮助文档：http://fishbase.readthedocs.io/
